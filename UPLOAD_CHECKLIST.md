## 🚀 GitHub へのアップロード手順

### **最簡単な方法：Web UI**

1. **GitHub にサインイン**
   - https://github.com にアクセス

2. **新しいリポジトリを作成**
   - 右上の「+」→「New repository」
   - **リポジトリ名**：`prompt-engineer`
   - **説明**：`AI Prompt Generator`
   - **Public** を選択（重要！）
   - 「Create repository」

3. **ファイルをアップロード**
   - 「Add file」→「Upload files」
   - 3つのファイルをドラッグ＆ドロップ
   - 「Commit changes」

---

## ⚙️ GitHub Pages を有効化

1. リポジトリ → 「Settings」
2. 左メニュー → 「Pages」
3. **Source** = 「Deploy from a branch」
4. **Branch** = 「main」/ **folder** = 「/ (root)」
5. 「Save」

---
